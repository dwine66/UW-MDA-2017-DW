Summary¶

  1. Stepwise Regression: The Stepwise regression eliminated engine location, fuel type, stroke, HP, and peak rpm as significant variables. The standard error and R^2 values (0.9566 vs. 0.9574) between Stepwise and standard models are nearly identical, so in this case it does not appear that Stepwise regression helps much.

  2. SVD: I tried 3 values of thres (thres = 50 is shown):

    For thres = 65 (all values) R^2 is 0.953 - nearly identical to the Stepwise analysis - but not better.

    For thres = 50 (near dropoff on mSVD plot), R^2 is basically unchanged.

    For thres = 13, R^2 is somewhat worse than the Stepwise (0.86 vs. 0.95) so that is probably too many.

    thres = 30 gives an R^2 of 0.91, so even there we are seeing significant degradation.

  3. Elastic Net:

    lambda = 5: R^2 = 0.824

    lambda = 10: R^2 = 0.937

    lambda = 15: R^2 = 0.956

    lambda = 20: R^2 = 0.958

It seems like a lambda of betweeen 10 and 15 is as good as we need here to converge, at least using R^2.

This model does as well as the others from an R^2 standpoint.

All three models appear to be about the same in their predictive capability, at least based on R^2.

# Plot function
plot.svd.reg <- function(df, k = 4,modelname){
  require(ggplot2)
  require(gridExtra)
  
  p1 <- ggplot(df) + 
    geom_point(aes(score, resids), size = 2) + 
    stat_smooth(aes(score, resids)) +
    ggtitle(paste('Residuals vs. fitted values: ',modelname))
  
  p2 <- ggplot(df, aes(resids)) +
    geom_histogram(aes(y = ..density..)) +
    geom_density(color = 'red', fill = 'red', alpha = 0.2) +
    ggtitle(paste('Histogram of residuals: ',modelname))
  
  qqnorm(df$resids,main=paste("Normal Q-Q Plot: ",modelname))
  
  grid.arrange(p1, p2, ncol = 2)
  
  df$std.resids = sqrt((df$resids - mean(df$resids))^2)  
  
  p3 = ggplot(df) + 
    geom_point(aes(score, std.resids), size = 2) + 
    stat_smooth(aes(score, std.resids)) +
    ggtitle(paste('Standardized residuals vs. fitted values: ',modelname))
  print(p3) 
  
  n = nrow(df)
  Ybar = mean(df$price)
  SST <- sum((df$price - Ybar)^2)
  SSR <- sum(df$resids * df$resids)
  SSE = SST - SSR
  cat(paste('SSE =', as.character(SSE), '\n'))
  cat(paste('SSR =', as.character(SSR), '\n'))
  cat(paste('SST =', as.character(SSE + SSR), '\n'))
  cat(paste('RMSE =', as.character(SSE/(n - 2)), '\n'))
  
  adjR2  <- 1.0 - (SSR/SST) * ((n - 1)/(n - k - 1))
  cat(paste('Adjusted R^2 =', as.character(adjR2)), '\n')
}
